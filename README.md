# gerencia-gastos-react

Aplicativo criado com react-native cli
Utiliza os recursos de autenticação e Realtime databse do Firebase

## Instalando dependências e executando projeto

### Execute os comandos 
  - yarn install
  - react-native link
  - react-native run-android
